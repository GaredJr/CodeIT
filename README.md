# Eksamensprosjekt 05. - 09.06.2026


## IDE
Navn: ***CODEIT***
Lett tilgjengelig og lettvindt Reddit typ sosial platform spesifikt for koding.
"Subreddits" for forskjellige typer kodespråk og teknologier.
Kan legge ved kodesnippets, github repositories, osv.

### TEKNOLOGIER
 * En gammel laptop med ubuntu server boota, for å hoste nettsiden og ulike teknologier.
 * Cloudflare, for tunneling av siden for å gjøre den tilgjengelig på et domene, og utenfor det lokale nettverket. (OG SIKKER KOBLING TIL SIDEN, UTEN LEKKASJE AV HEMMELIGE KEYS OSV.)
 * Supabase for databaser og brukerlagring.
 * HTML / CSS, for frontend design
 * JS, for enkel backend
 * Flask for koblinger
 * GIT + SSH, for syncing av siden mellom mac og server
 * GITHUB + KANBAN, for planlegging og backup
 * Adobe Photoshop / Illustrator for design av logo (Ikke relevant lol)
 * CHATGPT, for hjelp med planlegging og enkle elementer for raskere utvikling.

### FORMÅL
Gjøre det mye lettere for uviklere å dele kunnskap og hjelpe hverandre, uten krav for pålogging, men muligheten til det. Alle kan poste spørsmål, dele repositories og kode, og bli kjent med nye utviklere som er interresserte i samme ting som deg.
